# 籃球攻守紀錄系統 — 使用說明

## 啟動
**雙擊 `start.command`**（第一次若被 macOS 擋下：對它按右鍵 →「打開」→「打開」）。
會自動開啟瀏覽器到校對頁。要關閉時，回到黑色終端機視窗按 `Ctrl + C`。

> 需要電腦有 Python 3（macOS 內建）。本系統零安裝、不需網路。

## 兩個畫面
- **校對頁** `http://localhost:8770/`：左圖右表逐張核對，按「✓ 確認這張」存入資料庫。
- **統計總表** `http://localhost:8770/summary`：自動彙整已確認的場次，可匯出 CSV（Excel 可開）。

## 標記顏色
- 🟨 黃：辨識沒把握，請確認
- 🟩 綠：此格紙上有劃記
- 🟥 紅：正字筆數與總得分對不上（可能三分球或誤判）
- 🟧 橘 `?`：可能三分球／不確定
改過的格子標記會自動消失。

## 重要：資料安全
- 每次修改都**即時寫入 `bball.db`**（本機 SQLite 資料庫），關掉瀏覽器或換新版校對頁都不會掉。
- 前端 HTML 是可隨時更新／丟棄的；已校對的資料永遠在 `bball.db`。
- 想備份：校對頁底部「下載完整資料備份」，或直接複製 `bball.db` 檔。

## 檔案說明
- `server.py` — 本地服務（API + 資料庫），請勿刪除。
- `校對.html` / `統計總表.html` — 前端頁面。
- `bball.db` — 你的資料（自動產生）。**最重要，建議定期備份。**
- `辨識DB.json` — 辨識草稿來源（首次啟動時灌入資料庫用）。
